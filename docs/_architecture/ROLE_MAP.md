# ROLE MAP

## Employee
- Work leads
- Draft outreach
- Use AI assistant
- Create notes / tasks / proposal drafts

## Manager
- Everything employee can do
- Reassign leads
- Monitor team
- Review analytics
- Approve flows if configured

## Owner
- Executive visibility
- Cross-location trends
- Financial / pipeline oversight

## Admin
- Connectors
- Prompt/template governance
- Routing rules
- Deployment / sync / orchestration
